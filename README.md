# Dig

**A place to bury ideas and dig them back up.**

Dig is a local-first desktop app registry for vibe coders. Jot an idea the moment
it lands, let it settle, and let Dig surface an old one when you have forgotten it.
Promote the ones worth building into apps, and keep a plain feature and bug sheet
for each — nothing more.

Dig is not a project manager. No priorities, no statuses, no dates on tasks, no
sprints, no assignees. Ever.

Everything stays on your machine. No accounts, no telemetry, no network calls.

> land with the first release.
